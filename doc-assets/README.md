# ApexDox Documentation Instructions

Updating ApexDox happens in three parts.

1. Run ApexDoxSetup.sh
	```
	. doc-assets/ApexDoxSetup.sh
	```

1. Run ApexDox
	```
	SHIFT + COMMAND + P, select ApexDox: Run
	```

1. Run ApexDox cleanup
	```
	npm run prettier:docs
	```
